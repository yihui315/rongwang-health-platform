# 荣旺跨境健康平台 - 合规说明与产品文案边界

## 1. 文档目的

本文件用于约束平台前台页面、商品详情页、FAQ、AI 生成内容与运营素材的合规边界。

## 2. 核心原则

### 原则 A：不得将保健品或相关商品表述为药品
禁止出现：
- 治疗
- 治愈
- 根治
- 医治
- 临床证明可治疗
- 替代处方药
- 抗衰老神药
- 百病可用

### 原则 B：必须保留免责声明
涉及保健品时，默认必须出现：
- 本品不能替代药物

涉及跨境商品时，默认必须出现：
- 本商品符合原产国标准，可能与中国相关标准存在差异，请消费者知悉后谨慎选购

### 原则 C：所有 AI 输出都不是最终发布内容
AI 生成内容只能进入：
- 草稿
- 待审核
- 风险预检

不能直接默认上线。

## 3. 前台页面必须包含的合规信息

### 产品详情页必须包含
- 商品名称
- 原产地
- 规格 / 成分信息
- 中文说明占位
- 免责声明
- 联系方式
- 售后与投诉入口

### 合规说明页必须包含
- 跨境商品说明
- 标准差异提示
- 保健品免责声明
- 用户须知
- 联系我们

## 4. AI 内容生成红线
- 不能写疾病治疗承诺
- 不能给个体医疗建议
- 不能写三天见效、一吃就好、根治等表述
- 不能暗示替代医生或处方药

## 5. 审核状态建议
- draft
- generated
- compliance_flagged
- pending_manual_review
- approved
- rejected

只有 approved 才可以进入前台正式展示区。

## 6. 高风险类目提示
以下类目默认高风险，必须人工确认：
- NMN
- 涉激素类暗示
- 减肥强功效类
- 壮阳/性功能类
- 抗衰老强宣称类
- 带“药效”“治疗”暗示的文案

## 7. 页面标准声明模板

### 模板 1：保健品声明
本品为健康相关商品信息展示内容，不替代药物或医疗建议。具体使用请结合产品说明并咨询专业人士。

### 模板 2：法定提示
本品不能替代药物。

### 模板 3：跨境标准差异提示
本商品符合原产国标准，可能与中国相关标准存在差异，请消费者在充分了解后谨慎选购。
