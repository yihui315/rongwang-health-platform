# 荣旺跨境健康平台 - 12 周路线图

## 总体策略

前 12 周目标不是“做全”，而是“跑通”：
- 跑通站点
- 跑通商品导入
- 跑通 AI 内容生成
- 跑通合规预检
- 跑通基础推广与数据记录准备

## 第 1 月：搭站 + 合规 + 内容基础

### 第 1 周：站点搭建与法规模块准备
- 初始化项目
- 建立首页、产品页、合规页
- 明确免责声明模板
- 梳理 ICP、SSL、支付、CDN 需求

### 第 2 周：内容与 SEO 准备
- 完成 FAQ 页面
- 建立产品文案字段模型
- 定义 SEO 标题与关键词生成规则
- 建立内容审查规则

### 第 3 周：测试商品导入
- 完成商品导入 API
- 支持样例 URL 校验
- 建立抓取任务状态机
- 完成标准化商品对象

### 第 4 周：KOL / 直播 / 社媒准备位
- 预留推广入口
- 建立 campaign 数据表
- 定义后续社媒投放字段
- 准备活动与渠道映射

## 第 2 月：Agent 链路打通

### 第 5 周：内容生成 Agent
- 完成标题、描述、SEO 关键词生成
- 完成 FAQ 草稿生成
- 输出结构化内容对象

### 第 6 周：合规预检与人工审核
- 完成敏感词扫描
- 完成免责声明检查
- 建立人工审核状态
- 高风险内容阻断发布

### 第 7 周：物流与履约说明模块
- 上线物流说明页面
- 明确保税 / 直邮文案模板
- 准备售后与退换货说明

### 第 8 周：数据回看与第一轮迭代
- 查看导入与生成数据质量
- 修正文案模板
- 修正数据结构问题
- 补足错误日志字段

## 第 3 月：上架准备 + 指标化管理

### 第 9 周：上架 Agent 骨架
- 定义 listing payload
- 建立渠道映射
- 记录上架状态

### 第 10 周：KPI 与 A/B 测试基础
- 建立 campaign_stats 数据表
- 定义 CTR、CVR、ROI 基础字段
- 预留素材版本控制

### 第 11 周：整体打磨
- 修复页面问题
- 补移动端体验
- 补错误提示
- 强化审计与日志

### 第 12 周：复盘与下一阶段规划
- 总结已完成能力
- 总结合规与技术风险
- 输出 Phase 2 计划
