# 荣旺健康 RongWang Health

AI驱动的跨境保健品电商平台 — 香港荣旺健康科技有限公司

## 技术栈
- Frontend: Next.js 15 + React 19 + TypeScript + Tailwind CSS
- AI Engine: 一辉智能体 (Claude + MiniMax + Copilot + Qwen3)
- Database: Supabase (PostgreSQL)

## 核心模块
- AI健康测验 → 个性化方案推荐
- 4大健康方案 (抗疲劳/助眠/免疫/抗压)
- 购物车 & 结账
- 内容中心 (SEO文章)
- 家庭健康管理
- 订阅制服务

## 开发命令
```bash
npm run dev    # 开发服务器
npm run build  # 生产构建
npm start      # 生产运行
```
