🎨 全局视觉语言 · Style Guide

摄影风格

Clinical-premium — 融合医学专业感与生活方式美学。干净的纯色/渐变背景，柔和的漫射光，微妙的阴影。介于药品广告的严谨和DTC品牌的温暖之间。

色温与光线

偏冷的自然光为主基调（5600K-6500K），用暖色点缀（橙色CTA、琥珀色胶囊）。产品图使用3/4侧光+底部反射。人物场景使用逆光/窗光。

构图原则

留白大方（产品主体占画面40-60%），遵循三分法，产品始终清晰锐利。避免拥挤排版。背景简洁，可用渐变或虚化自然元素。

人物方向

亚洲面孔为主（25-45岁），健康活力但不夸张。避免欧美健身房风格，偏向都市白领/职业女性/年轻父母场景。表情自然不造作。

Midjourney 通用参数

所有 prompt 使用 --v 6.1，产品图 --s 200-300（精致），场景图 --s 400-600（自然美）。--no text, logo, watermark, label 固定后缀。

输出格式

Hero/Banner: --ar 16:9 或 --ar 3:2。产品图: --ar 1:1 或 --ar 4:5。Scene卡片: --ar 4:3。文章头图: --ar 16:9。

🔄 生成工作流程

Step 1: 批量生成

按分类顺序逐个复制 prompt 到 Midjourney。建议使用 /imagine 命令，一次生成4张变体。每个 prompt 预计生成时间 30-60s。

Step 2: 选图 + Upscale

从4张变体中选择最佳构图，点击 U1-U4 进行 Upscale。产品图选 Upscale (Subtle) 保持细节，场景图可选 Upscale (Creative)。

Step 3: 后处理

MJ 可能自动添加文字/logo，需在 Photoshop 或 Canva 中移除。产品图建议用 Remove.bg 去背后放在网站渐变背景上。

Step 4: 命名 + 部署

按 类型-编号.webp 命名（如 hero-01.webp、scene-fatigue.webp）。转换为 WebP 格式压缩到 200KB 以下。上传到 /images/ 目录。

Step 5: 变体调优

如某张图效果不理想，可用 --seed 锁定好的构图再调整描述词。或使用 /blend 混合两张接近的变体。也可用 Vary (Subtle/Strong)。

Step 6: 代码集成

将图片放入站点后，告诉我，我会将所有 V2 页面中的 emoji 占位符和渐变色块替换为真实图片，并添加 loading="lazy" 和响应式 srcset。
